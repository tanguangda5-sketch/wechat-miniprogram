# 软著材料目录

本目录用于存放“农旅e站”软件著作权申请相关草稿材料，建议在最终提交前根据实际申请主体、软件名称、版本号与发布日期再做一次校对。

## 文件说明

1. `农旅e站-软件使用说明书.md`
   软件说明书正文草稿，可直接补充截图后导出为 PDF。
2. `农旅e站-源代码交存说明.md`
   说明如何整理源代码并制作“前 30 页 + 后 30 页”的鉴别材料。
3. `农旅e站-软著申请材料清单.md`
   用于核对申请表、身份证明、源程序材料、文档材料及其他证明文件。

## 建议使用顺序

1. 先修改《软件使用说明书》中的软件名称、版本号、完成日期。
2. 再按《源代码交存说明》整理代码并生成 PDF。
3. 最后按《软著申请材料清单》逐项检查并上传。

## 导出建议

1. 优先导出为 PDF。
2. 页面统一使用 A4。
3. 标题层级尽量清晰。
4. 软件说明书建议补充关键页面截图。
5. 源代码鉴别材料建议显示文件路径、页码与等宽字体。
